# AI Based FAQ Chatbot

This is an AI-based FAQ Chatbot built using Python, NLP, and Streamlit.

## Features
- Answers frequently asked questions
- Uses NLP for text similarity
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- scikit-learn
- JSON

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
